# Industrial Bearing & Chain Centre (IBCC)

This is the official front-end repository for the **Industrial Bearing & Chain Centre (IBCC)** website. It is built to be an extremely fast, SEO-optimized, mobile-responsive catalog and lead-generation site.

## Tech Stack
* **Framework:** [Next.js 14+ (App Router)](https://nextjs.org/)
* **Language:** TypeScript
* **Styling:** Tailwind CSS + Custom Vanilla CSS (for complex animations and gradients)
* **Icons:** Inline SVG Icons (No external font-awesome libraries to maximize loading speed)
* **Deployment:** Vercel

## Getting Started

First, install the dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## вљ пёЏ Important: How to Add New Products

To make managing hundreds of products easy, the product database (`src/data/products.ts`) is **auto-generated** based on the images placed in the public folder.

**Do NOT manually edit `src/data/products.ts`.** If you do, your changes will be overwritten.

### Steps to add a new product:
1. Obtain the new product image.
2. Rename the image accurately. The name of the image file will become the name of the product on the website.
3. Place the image inside `public/images/real image/`.
4. Run the rebuild script from the terminal:
   ```bash
   node rebuild_products.js
   ```
5. The script will scan the folder, categorize the product using keyword matching, format the title, and update `src/data/products.ts`. 

## Global CSS & Styling Guidelines

* **Colors:** Brand colors (`--navy`, `--amber`, etc.) are defined in `src/app/globals.css`. 
* **Animations:** Performance-heavy animations (like the background blur orbs) are disabled or optimized for mobile viewports using `@media (max-width: 768px)` in `globals.css` to prevent scroll lag on low-end Android devices.
* **Layouts:** Use `container-xl` class for consistent horizontal padding and max-width across pages.
* **SVH Height:** The homepage hero uses `min-height: 92svh` to prevent layout jumping on mobile browsers when the address bar hides/shows.

## SEO & Analytics Setup

* **Sitemap & Robots.txt:** Handled dynamically via `src/app/sitemap.ts` and `public/robots.txt`.
* **Meta Tags:** Global default OpenGraph and Twitter tags are defined in `layout.tsx`. Each page (like `/gallery` or `/about`) exports its own specific `metadata` object.
* **JSON-LD Schema:** `layout.tsx` contains LocalBusiness structured data to help IBCC rank highly on Google Maps and localized search results for Hubballi.

## Environment Variables
If running on Vercel, ensure the following environment variables are set in the dashboard:
* `NEXT_PUBLIC_SITE_URL` = `industrialbearingcentre.in` (Used for generating absolute URLs in the sitemap).
